Writable Store


              import { writable } from 'svelte/store'

              const fruitStore = writable<string[]>([])

              // subscribe to updates
              fruitStore.subscribe(fruits => {
                console.log(fruits)
              })

              // set the value of the store
              fruitStore.set(['apple', 'banana', 'pear'])

              // update the value in the context of existing values
              fruitStore.update(currentFruits => {
                return [...currentFruits, 'grapes']
              })

Custom Stores and Hooks


              import { onMount } from 'svelte'
              import { writable } from 'svelte/store'

              export default function useElapsed() {
                const { subscribe, update } = writable(0)

                onMount(() => {
                  const interval = setInterval(() => {
                    update(elapsed => elapsed + 1)
                  }, 1000)

                  return () => clearInterval(interval)
                })

                return { subscribe }
              }

Transitions

  • 7 built-in transitions:
    fade, blur, fly, slide, scale, draw, crossfade
  • In and out transitions can be declared separately
  • Transitions can be parametrized
  • Custom transitions

              <p in:fly="{{ y: 200, duration: 2000 }}" out:fade>
                Flies in, fades out
              </p>
